In one sentence: This is a custom AI-powered CRM + project delivery system designed specifically for real estate media operations.

1. End-to-End Workflow

Website visitors can enter through AI chat, package builder, or contact forms. All paths feed one CRM workflow.
  • Client lands on website and interacts with AI assistant or Build My Package.
  • System captures requirements, contact details, property type, and service needs.
  • Lead is automatically categorized by source: AI Assistant, Packages, or Website Form.
  • Admin team reviews lead, updates status, and manages follow-ups from CRM panel.
  • Quote is generated and tracked through conversion statuses.
  • Once approved, project moves through delivery pipeline stages.
  • Invoices and client communication are managed in the same platform.
  • Client portal shows project updates, invoices, requests, and messages.

2. Core Features Available Now

A) Website, AI, and Lead Capture

  • AI assistant qualifies leads in natural conversation (EN/FR behavior supported by language toggle).
  • Package builder popup with preset packages and custom package mode.
  • Live estimate card appears in AI chat with selected options and total price.
  • AI estimate actions:
    • Use this estimate in Build My Package (prefilled, faster conversion)
    • Edit options before submit (opens directly to editing step)
  • Automatic lead intake from all sources into CRM.
  • Dedicated lead tabs: Lead Pipeline (All), Leads from AI Assistant, Leads from Packages.

B) Sales Pipeline and Conversion

  • Quote workflow: New, Reviewed, Contacted, Booked, Lost.
  • Admin quote detail page with timeline, manual line-item editor, and resend email action.
  • User quote detail + revision request flow with timeline logging.
  • Conversion analytics widgets and funnel charts on admin dashboard.

C) Invoices and Financial Tracking

  • Dedicated invoice module with status filters and paid/unpaid visibility.
  • Invoice statuses: Draft, Sent, Partial, Paid, Overdue.
  • Invoice status updates trigger client-side notifications.

D) Project Delivery Management

  • Projects tab with ongoing vs completed visibility.
  • Kanban + table views for operational control.
  • Delivery stages: Accepted, Shooting, Editing, Complete.
  • Overdue and follow-up indicators for execution discipline.

E) Accounts, Roles, and Security

  • Roles supported: Owner/Admin, Manager, Photographer, Editor, Client/Agent.
  • Role-based permission enforcement on routes and actions.
  • Read-only protection for restricted roles on lead/quote updates.
  • Admin can create/delete users and client accounts.

F) Notifications and Client Communication

  • Top-right notification center with unread count.
  • Category filters: All, Quotes, Invoices, Messages.
  • Events covered: quote changes, invoice actions, project updates, service requests.
  • Centralized client timeline and messaging history.

3. Admin and Client Portals

Admin Portal Lead operations, quote pipeline, invoice control, project board, user management, analytics dashboard, and notifications.
Client Portal View quotes, projects, invoices, request services, and track communication history in one place.
Mobile Support Responsive panel behavior for mobile devices, including sidebar overlay, table scroll handling, and mobile-friendly controls.
