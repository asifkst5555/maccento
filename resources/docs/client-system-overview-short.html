Maccento AI Client Portal

What it is: A custom AI-powered platform that captures leads, manages quotes/invoices, and tracks project delivery in one place.

Main result: More conversions, less manual follow-up, and a better client experience.

How It Works (Simple)

  • Client chats with AI assistant or uses Build My Package.
  • System auto-captures lead details and source.
  • Admin team manages lead status, quote pipeline, and follow-ups.
  • Approved deals move into project workflow and invoicing.
  • Client portal provides updates, invoices, and communication history.

Top Features Included

Lead Capture AI chat, package popup, website form intake, and separate source tabs (AI / Packages / All).
Sales Workflow Quote statuses (new, reviewed, contacted, booked, lost), revision requests, and conversion charts.
Invoices Dedicated invoice panel with paid/unpaid tracking and status workflow.
Project Delivery Table + Kanban board (Accepted, Shooting, Editing, Complete) with ongoing/completed visibility.
Team and Roles Role-based access for Admin, Manager, Photographer, Editor, and Client/Agent.
Notifications Live notification center with filters (All, Quotes, Invoices, Messages).

Business Value

For Maccento: Faster response times, better pipeline control, and easier scaling with less manual admin work.

For clients: Faster booking flow, clearer communication, and a professional portal experience.
